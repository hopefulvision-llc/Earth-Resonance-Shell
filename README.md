### C. Fibonacci Sphere Algorithm
```python
def fibonacci_sphere(n=72):
    points = []
    phi = (1 + sqrt(5)) / 2
    for i in range(n):
        theta = i * 137.5 * pi / 180  # Golden angle in radians
        phi_angle = acos(1 - 2 * i / (n - 1))  # Polar angle
        x = sin(phi_angle) * cos(theta)
        y = sin(phi_angle) * sin(theta)
        z = cos(phi_angle)
        points.append((x, y, z))
    return points
```
